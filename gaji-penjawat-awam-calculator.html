Instructions

How to use it

Enter gaji pokok, elaun, COLA, elaun lain, and potongan.

Formula used

Gaji bersih = gaji pokok + elaun - potongan.

Example calculation

If gaji kasar is RM 4,000 and deductions are RM 500, gaji bersih is RM 3,500.
