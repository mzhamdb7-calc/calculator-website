    Currency Converter

    Enter the exchange rate manually for this static GitHub Pages converter.

    Because this is a static site, live exchange rates need a backend/API. This page uses the rate you enter.

    Input details

    What does this calculator do?

    Use this to convert an amount from one currency to another using the exchange rate entered in the form.

    Instructions

    How to use it

    Enter the amount, choose the from/to currencies, and enter or confirm the exchange rate.

    Formula used

    Converted amount = original amount × exchange rate.

    Example calculation

    If RM 100 is converted at a rate of 0.21, the result is 21 in the target currency.
