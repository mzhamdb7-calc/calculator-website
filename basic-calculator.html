    Basic Calculator

    Free online calculator for quick math calculations.

    What does this calculator do?

    Use this for quick everyday maths such as addition, subtraction, multiplication, division, powers, and square root.

    Instructions

    How to use it

    Type or tap the numbers, choose the operator, then press =. Use AC to clear and ← to delete one character.

    Formula used

    It follows the normal order of operations: brackets and powers first, then multiplication/division, then addition/subtraction.

    Example calculation

    8 + 2 × 3 = 14 because multiplication is calculated before addition.

    References

    Order of operations

    Standard arithmetic uses a fixed order so calculations are consistent.
