    What does this calculator do?

    Use this to convert common units such as length, weight, temperature, area, volume, speed, currency, data storage, time, and energy/power.

    Instructions

    How to use it

    Choose the conversion type, enter the value, then choose the starting unit and target unit.

    Formula used

    The calculator uses fixed conversion factors for physical units. Currency mode fetches live daily exchange rates from the web.

    Example calculation

    1 kilometre = 1000 metres. 1 MB = 1000 KB. Currency conversion fetches live daily exchange rates from the web when Currency is selected.
