    Inflation Calculator

    See how inflation changes future cost and buying power.

    Input details

    What does this calculator do?

    Use this to estimate how inflation changes buying power or future cost over time.

    Instructions

    How to use it

    Enter the starting amount, inflation rate, and number of years.

    Formula used

    Future value = present value × (1 + inflation rate) ^ years.

    Example calculation

    RM 1,000 at 3% inflation for 5 years becomes about RM 1,159.

    References

    Inflation estimate

    Inflation is usually measured with price indexes and can vary by country and time period.

    General note

    This is a simplified estimate. Real inflation differs by item, location, and year.
